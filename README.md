# Grandata – Fondazione CRC

Repository ufficiale open data della Fondazione CRC.

## Struttura
- /datasets → dataset CSV
- /docs → documentazione

## Linee guida fornitore
- file CSV formato UTF-8
- struttura per cartelle tematiche
- ogni dataset deve avere README.md

## Automazione
I file vengono aggiornati automaticamente.
